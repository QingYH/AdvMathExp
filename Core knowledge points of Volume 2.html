🧊 空间解析几何

向量 · 平面 · 曲面 · 柱面 · 锥面 · 旋转曲面
📌 向量的数量积与向量积 基本运算

数量积（点积）：$\mathbf{a} \cdot \mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\theta = a_1b_1 + a_2b_2 + a_3b_3$，用于求夹角、投影、判定垂直。

向量积（叉积）：$\mathbf{a} \times \mathbf{b} = (a_2b_3-a_3b_2,\; a_3b_1-a_1b_3,\; a_1b_2-a_2b_1)$，模长 $|\mathbf{a} \times \mathbf{b}| = |\mathbf{a}||\mathbf{b}|\sin\theta$，方向垂直a,b所在平面，用于求法向量、面积。

📝 常见题型：求夹角、单位法向量、平行四边形面积 选择题 计算题
📌 空间直线与平面方程 几何表示

直线：对称式 $\frac{x-x_0}{m} = \frac{y-y_0}{n} = \frac{z-z_0}{p}$，参数式 $x=x_0+mt,\; y=y_0+nt,\; z=z_0+pt$，一般式（两平面交线）。

平面：点法式 $A(x-x_0)+B(y-y_0)+C(z-z_0)=0$，一般式 $Ax+By+Cz+D=0$，截距式 $\frac{x}{a}+\frac{y}{b}+\frac{z}{c}=1$。

📝 常见题型：求直线/平面方程、位置关系（平行/垂直/夹角）、距离 综合题 证明题
📌 二次曲面 · 柱面 · 锥面 · 旋转曲面 标准方程

椭球面：$\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1$；双曲面：$\frac{x^2}{a^2}+\frac{y^2}{b^2}-\frac{z^2}{c^2}=\pm 1$；抛物面：$z = \frac{x^2}{a^2}+\frac{y^2}{b^2}$（椭圆抛物），$z = \frac{x^2}{a^2}-\frac{y^2}{b^2}$（双曲抛物）。

柱面：方程中缺少一个变量，例如 $x^2 + y^2 = R^2$（圆柱面），$y^2 = 2px$（抛物柱面），$\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$（椭圆柱面）。

锥面：$\frac{x^2}{a^2}+\frac{y^2}{b^2} = \frac{z^2}{c^2}$（圆锥面）。

旋转曲面：由曲线绕某轴旋转形成。如 $xOz$ 面上曲线 $z=f(x)$ 绕 $z$ 轴旋转：$z = f(\sqrt{x^2+y^2})$；绕 $x$ 轴旋转：$z = \pm f(\sqrt{y^2+z^2})$。

📝 常见题型：由方程识别曲面类型、柱面/锥面/旋转曲面方程、截痕分析 填空题 选择题

🌀 多元函数微分学

连续·偏导·链式法则
📌 连续、偏导、可微的关系 概念辨析

关系：可微 ⇒ 连续；可微 ⇒ 偏导存在；偏导存在且连续 ⇒ 可微。反之均不成立。

全微分：$df = f_x(x_0,y_0)\,dx + f_y(x_0,y_0)\,dy$（若可微）。

方向导数与梯度：梯度 $\nabla f = (f_x, f_y, f_z)$，方向导数 $D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u}$（$\mathbf{u}$为单位向量），梯度方向为函数增长最快方向。

📝 常见题型：判定可微性、求全微分/梯度/方向导数、关系反例 证明题 计算题 辨析题
📌 复合函数链式法则 常见形式

形式一（一元复合）：$z = f(u,v),\; u=u(t),\; v=v(t) \;\Rightarrow\; \frac{dz}{dt} = f_u \cdot u'(t) + f_v \cdot v'(t)$

形式二（多元复合）：$z = f(u,v),\; u=u(x,y),\; v=v(x,y) \;\Rightarrow\; z_x = f_u u_x + f_v v_x,\; z_y = f_u u_y + f_v v_y$

形式三（全微分形式不变性）：$dz = f_u\,du + f_v\,dv$，其中 $du = u_x dx + u_y dy$。

隐函数求导：$F(x,y)=0 \;\Rightarrow\; \frac{dy}{dx} = -\frac{F_x}{F_y}$；$F(x,y,z)=0 \;\Rightarrow\; z_x = -\frac{F_x}{F_z},\; z_y = -\frac{F_y}{F_z}$。

📝 常见题型：复合函数求导、隐函数偏导、全微分计算 高频考点 证明题 计算题
📌 切平面·法线·极值 几何应用

曲面的切平面与法线：曲面 $F(x,y,z)=0$ 在点 $P_0$ 处切平面 $F_x(P_0)(x-x_0)+F_y(P_0)(y-y_0)+F_z(P_0)(z-z_0)=0$，法线 $\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}$。

无条件极值：驻点 $f_x=0, f_y=0$，用 $\Delta = f_{xx}f_{yy} - (f_{xy})^2$ 判定。

条件极值（拉格朗日乘数法）：求 $f(x,y)$ 在约束 $g(x,y)=0$ 下的极值，构造 $L = f + \lambda g$，解 $L_x=0,\; L_y=0,\; g=0$。

📝 常见题型：切平面/法线方程、极值点与极值、条件极值应用题 应用大题 最值问题 证明题

📦 重积分

二重 · 三重 · 技巧
📌 二重积分 · 两种区域与交换次序 X型 · Y型

X型区域：$D = \{ (x,y) \mid a \le x \le b,\; y_1(x) \le y \le y_2(x) \}$，积分 $\int_a^b dx \int_{y_1(x)}^{y_2(x)} f(x,y)\,dy$。

Y型区域：$D = \{ (x,y) \mid c \le y \le d,\; x_1(y) \le x \le x_2(y) \}$，积分 $\int_c^d dy \int_{x_1(y)}^{x_2(y)} f(x,y)\,dx$。

交换积分次序技巧：①画出区域草图；②根据区域类型重新划分积分限；③注意分段区域需要拆分。

极坐标：$x = r\cos\theta,\; y = r\sin\theta,\; d\sigma = r\,dr\,d\theta$，$\iint_D f(x,y)\,d\sigma = \int_\alpha^\beta \int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta, r\sin\theta)\, r\,dr\,d\theta$。

📝 常见题型：交换积分次序、极坐标化、计算面积/体积 计算题 填空题
📌 三重积分 · 穿针法与切片法 计算技巧

穿针法（先一后二）：$\iiint_\Omega f\,dV = \iint_{D_{xy}} \left[ \int_{z_1(x,y)}^{z_2(x,y)} f(x,y,z)\,dz \right] dxdy$，适用于z方向有明确上下界。

切片法（先二后一）：$\iiint_\Omega f\,dV = \int_{z_1}^{z_2} \left[ \iint_{D_z} f(x,y,z)\,dxdy \right] dz$，适用于截面面积易求。

柱坐标：$x = r\cos\theta,\; y = r\sin\theta,\; z = z$，$dV = r\,dr\,d\theta\,dz$；球坐标：$x = \rho\sin\phi\cos\theta,\; y = \rho\sin\phi\sin\theta,\; z = \rho\cos\phi$，$dV = \rho^2\sin\phi\,d\rho\,d\phi\,d\theta$。

📝 常见题型：坐标变换、穿针法/切片法应用、计算体积/质量 综合大题 应用题

🌀 曲线·曲面积分

格林 · 高斯 · 计算方法
📌 两类曲线积分 · 格林公式 计算方法与场景

第一类（对弧长）：$\int_L f(x,y)\,ds$
计算方法：参数化 $x=x(t), y=y(t), \alpha \le t \le \beta$，则 $ds = \sqrt{[x'(t)]^2 + [y'(t)]^2}\,dt$，积分化为 $\int_\alpha^\beta f(x(t),y(t)) \sqrt{[x'(t)]^2 + [y'(t)]^2}\,dt$。

第二类（对坐标）：$\int_L P\,dx + Q\,dy$
计算方法：参数化 $x=x(t), y=y(t)$，则 $dx = x'(t)dt, dy = y'(t)dt$，积分化为 $\int_\alpha^\beta [P(x(t),y(t))x'(t) + Q(x(t),y(t))y'(t)]\,dt$。

格林公式：$\oint_C P\,dx + Q\,dy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) d\sigma$（正向边界）。

📌 适用场景与做法
① 曲线积分为封闭曲线 → 直接使用格林公式化为二重积分计算。
② 曲线积分路径不封闭 → 添加辅助弧段或直线段使其封闭，应用格林公式后减去补加路径的积分。
③ 若 $\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$（积分与路径无关）→ 选取简单路径（如平行于坐标轴的折线）计算，或找出势函数 $u$ 使 $du = Pdx + Qdy$，则 $\int_L Pdx+Qdy = u(B)-u(A)$。
④ 求区域面积：$A = \frac12\oint_C x\,dy - y\,dx$。

📝 常见题型：曲线积分计算、格林公式应用（求面积、积分与路径无关、补线法） 技巧型 计算题
📌 两类曲面积分 · 高斯公式 计算方法与场景

第一类（对面积）：$\iint_\Sigma f(x,y,z)\,dS$
计算方法：显式曲面 $z=z(x,y)$，$dS = \sqrt{1+z_x^2+z_y^2}\,dxdy$，积分化为 $\iint_{D_{xy}} f(x,y,z(x,y)) \sqrt{1+z_x^2+z_y^2}\,dxdy$。

第二类（对坐标）：$\iint_\Sigma P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy$
计算方法：显式曲面 $z=z(x,y)$，取上侧时 $\iint_\Sigma R\,dx\,dy = \iint_{D_{xy}} R(x,y,z(x,y))\,dxdy$（下侧取负）。其他项类似，需根据曲面方向投影到相应坐标平面。

高斯公式：$\displaystyle \iint_\Sigma P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy = \iiint_\Omega \left( \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} \right) dV$（外侧）。

📌 适用场景与做法
① 曲面封闭 → 将曲面积分化为三重积分计算。
② 若散度 $\nabla \cdot \mathbf{F} = 0$，则通量为0。
③ 曲面不封闭 → 补面（添加辅助曲面）使其封闭，再应用高斯公式，最后减去补面的积分。
④ 注意曲面方向：外侧为正，内侧为负。

📝 常见题型：曲面积分计算、高斯公式求通量、散度应用、补面技巧 综合应用 证明题

📈 无穷级数

收敛 · 幂级数 · 傅里叶
📌 数项级数 正项 · 交错 · 绝对收敛

正项级数：比较判别法、比值（达朗贝尔）判别法 $\lim_{n\to\infty} \frac{a_{n+1}}{a_n} = \rho$（$\rho<1$收敛，$\rho>1$发散），根值判别法。

交错级数：莱布尼茨判别法（单调递减趋于0则收敛）。

绝对收敛：$\sum |a_n|$ 收敛 ⇒ 绝对收敛；若 $\sum a_n$ 收敛而 $\sum |a_n|$ 发散，则条件收敛。

📝 常见题型：判别收敛性、绝对/条件收敛、p级数/等比级数 证明题 选择题 判别题
📌 幂级数 收敛域 · 和函数 · 展开

收敛半径：$R = \frac{1}{\lim\limits_{n\to\infty} \sqrt[n]{|a_n|}}$ 或 $R = \lim\limits_{n\to\infty} \left| \frac{a_n}{a_{n+1}} \right|$，收敛区间 $(x_0-R,\; x_0+R)$，端点需单独判定。

和函数：在收敛域内，幂级数可逐项求导/积分，常见展开式 $e^x = \sum_{n=0}^\infty \frac{x^n}{n!}$，$\frac{1}{1-x} = \sum_{n=0}^\infty x^n$。

间接展开：利用已知展开式（如 $\sin x, \cos x, \ln(1+x)$）通过变量代换、四则运算、求导/积分得到。

📝 常见题型：求收敛半径/收敛域、求幂级数和函数、函数展开为幂级数 综合大题 证明题 计算题
📌 傅里叶级数 周期 2π · 完整公式

傅里叶系数：
$a_0 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\,dx$，
$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos(nx)\,dx \quad (n=1,2,\dots)$，
$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin(nx)\,dx \quad (n=1,2,\dots)$。

傅里叶级数：$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^\infty \left( a_n \cos(nx) + b_n \sin(nx) \right)$。

收敛定理：若f以$2\pi$为周期且分段光滑，则傅里叶级数收敛于 $\frac{f(x^+)+f(x^-)}{2}$。

奇偶延拓：奇函数展开为正弦级数（$a_n=0$），偶函数展开为余弦级数（$b_n=0$）。

📝 常见题型：求傅里叶系数、写出级数、利用收敛定理求和、奇偶延拓 填空题 选择题 计算题
